# autoupgrade-linux

Skrip konfigurasi update repositori dan upgrade packages secara otomatis

Cara jalanin:

Versi curl:

```
curl -sL https://raw.githubusercontent.com/Kornel-Y/autoupgrade-linux/refs/heads/main/config.sh | sudo bash
```

Versi wget:

```
wget -qO- https://raw.githubusercontent.com/Kornel-Y/autoupgrade-linux/refs/heads/main/config.sh | sudo bash
```
